# Ma Collection de Musique

Site web personnel listant ma collection de disques (CD, Vinyles, etc.) récupérée depuis Discogs.

## Fonctionnalités
- Liste complète avec recherche et filtres.
- Mise à jour automatique quotidienne via GitHub Actions.
- Hébergé gratuitement sur GitHub Pages.

## Installation
Ce projet est déployé automatiquement.
